Trippy - Travel Planning

This was my first project and it help make travel planning easier by giving them a place to plan their trip and keep track of additional notes that they have relating to each location.

Tech Stack

The program was built using RealmSwift Database, UIKit, SwiftUI, and the Yelp API. The programming language used was Swift.
